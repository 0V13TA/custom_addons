The Custom Branding settings are located in **Odoo Settings**, not in a separate menu.

### Step-by-Step:

1. **Click the Settings icon** (âš™ï¸) in the top-right corner of Odoo
   - Or go to: **Apps > Settings** from the main menu

2. **Look for "Custom Branding"** in the left sidebar
   - It should appear as a separate app/section
   - If you don't see it, scroll down the left sidebar

3. **Click on "Custom Branding"** to see all the branding options

## If You Don't See "Custom Branding":

### Option 1: Module Not Installed
1. Go to **Apps** menu
2. Click **"Update Apps List"** (top right)
3. Search for **"Custom Branding"**
4. Click **Install**

### Option 2: Module Needs Upgrade
1. Go to **Apps** menu
2. Search for **"Custom Branding"**
3. If it shows "Installed", click **Upgrade**

### Option 3: Check Module Status
1. Go to **Apps** menu
2. Remove the "Apps" filter (click "All")
3. Search for **"Custom Branding"**
4. Check if it's installed and active

## Alternative: Direct URL

You can also access Settings directly:
- URL: `http://localhost:8072/web#action=base.action_res_config_settings`

Then look for "Custom Branding" in the left sidebar.

## What You Should See

Once you access Settings > Custom Branding, you'll see:
- Database Page Logo (upload field)
- Database Page Title (text field)
- Login Page "Powered by" Text (text field)
- Login Page "Powered by" URL (text field)
- POS Receipt Footer Text (text field)
